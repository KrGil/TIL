# [SpringBoot] Gradle로 jar(war) 파일 빌드하고 실행하기

https://programmer-ririhan.tistory.com/120



### Windows

> ```gitbash``` , ```powershell```

```
./gradlew build
```

> ``` cmd```

```
gradlew build
```



### MacOS

```
./gradlew build
```



### 빌드 재빌드(완전히 삭제 후 다시 빌드)

```
./gradlew clean build
```
