# IP란 무엇일까

### 공인 IP(public)

- ISP(인터넷 서비스 공급자)가 제공하는 IP
- 공용 IP, 외부에 공개되어 있는 IP
- 전세계 유일한 IP 주소를 갖는다.
- 다른 PC로부터의 접근이 가능.



### 사설 IP(private)

- 일반 가정이나 회사 내에서 할당된 네트워크

- 로컬 IP, 가상 IP.

- 하나의 네트워크 안에서만 유일

    

#### 사설 IP 주소대역

Class A : 10.0.0.0 ~ 10.255.255.255

Class B : 172.16.0.0 ~ 172.31.255.255

Class C : 192.168.0.0 ~ 192.168.255.255
